-------  LIM MODELLING  --------

Two different models of the LIM are developed:

----------------------------------------------------------------------------------------

1- Lim model with the end effect following Field Oriented Control LIM (DOI:10.1049/ip-epa:20045185):
  
      Files: LIM_model_controller.slx 
             LIMvariables.m  -----> initialize variables and PID values

-------------------------------------------------------------------------------------------

2- Lim model with end effect in the model itself but not in the controller. 
      Based on: -FULLTEXT01.pdf for the modelling of the LIM
                -Control of Voltage-Source Converter and Variable-Speed Drives (for the design of the current and speed controllers). Chapter of AC motors.

  
      Files: model15062023.slx 
             LIM_parameters.m  -----> initialize variables and PID values
